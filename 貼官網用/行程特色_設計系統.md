# 行程特色｜視覺與文字設計系統（祕魯 55 周年）

本文件與 `行程特色_開發版.html`／`行程特色_貼上版.html` 內之 **CSS 變數、工具 class** 對齊；上線貼 CMS 時仍以 **貼上版全段** 為準。技術上限（`<style>` 數量、欄位截斷等）見 `行程特色_安全版_RULES.md`。

---

## 1. 文字階層（中文為主）

| 階層 | 用途 | HTML 建議 | Class（工具） |
|------|------|-----------|----------------|
| **超大標** | 主視覺一句話、最強聚焦 | 每區塊至多一個 `<h1>` 或語意對等 | `.pe55-type-xl`（常與 `.pe55-title` 併用） |
| **大標** | 區塊主標題 | `<h2>` | `.pe55-type-lg`（常與 `.pe55-h2` 併用） |
| **次標** | 區塊內分段、輔助標題 | `<h3>` | `.pe55-type-md` |
| **小標** | 卡片標、列表小標 | `<h4>` 或卡片內 `<h3>` | `.pe55-type-sm`（卡片內建議加於標題） |
| **內文** | 段落、清單、說明 | `<p>`、`<li>` | `.pe55-type-body`（或用區塊既有樣式，字級會跟隨變數） |

字級以 **`clamp()`** 實作，變數名稱：`--pe55-t-xl` … `--pe55-t-body`（定義於 `#pe55-cms-safe-root`）。

## 2. 字型

| 角色 | 字族 | 說明 |
|------|------|------|
| **介面／中文內文** | **Noto Sans TC**（Google Fonts，與 `@import` 一併載入） | 主閱讀字體；後備：`Microsoft JhengHei`、`PingFang TC`、`Microsoft YaHei`、sans-serif |
| **裝飾／拉丁語系** | **Cormorant Garamond**（serif） | 用於西班牙語標語、裝飾性西文標題等；**不建議**長篇中文全文使用 |

- **變數**：`--pe55-font-ui`、`--pe55-font-latin`
- **套用西文裝飾**：在元素上加 `.pe55-latin`，並建議加 **`lang="es"`**（或其他對應語言碼）以利無障礙與字型選擇。

**CMS 限制**：全頁僅 **一條** `@import`（已合併 Noto Sans TC + Cormorant Garamond），放在**唯一** `<style>` 區塊之最開頭；勿再新增第二個 `<style>` 或額外 `@import`。

若後台會**過濾或封鎖**對 `fonts.googleapis.com` 的 `@import`，網頁仍會以後備字族顯示（中文與西文皆降級為系統／Times），不影響結構；必要時改由官網主題載入字型或改 Canva 合成圖。

## 4. 圖騰感裝飾（抽象幾何）

避免直接使用具**特定宗教／族群神聖意涵**的符號當行銷圖騰；公開頁面以 **抽象階梯紋、細線、色帶** 表現安地斯織物與建築的**幾何節奏**即可。

| Class | 用途 |
|--------|------|
| `.pe55-motif-line` | 橫向短節奏線（金／陶土交替感） |
| `.pe55-band-andes` | 細色帶（藍→金漸層），作區塊頂／底裝飾 |
| `.pe55-card--pattern`（選用） | 卡片左側極淡階梯紋裝飾邊 |

複雜圖樣若需固定版式，可改以 **Canva 輸出單張圖** 嵌入（見 RULES「方法 A」與 Canva 小節），較省 CMS 內文長度。

## 6. 圖片馬賽克（視覺精選）

| Class | 說明 |
|--------|------|
| `.pe55-gallery-rail` | 全寬軌（`100vw`／`margin-left: calc(50% - 50vw)`），沙色底 |
| `.pe55-mosaic` | 主圖＋右側雙小圖之網格（窄螢幕改單欄） |
| `.pe55-ph` | 單格相框：圓角、陰影、底部漸層遮罩、`figcaption` 疊字 |
| `.pe55-mosaic-reveal` | 與 `.pe55-anim-onview` 併用：三格圖**依序**進場 |

**高解析載入**：主圖建議 `loading="eager"`、`fetchpriority="high"`，輔圖 `loading="lazy"`；`srcset` 多寬度 + `sizes` 對應欄寬；`q=88` 等參數可依圖床調整。示意圖來源可暫用 [Unsplash](https://unsplash.com/)（請遵守其授權與標示習慣）；**正式上線請改為貴社圖床**固定網址（見 RULES「方法 A」）。

**滑鼠／觸控**：僅在 `(hover: hover) and (pointer: fine)` 時放大圖片，減少手機誤觸；`prefers-reduced-motion: reduce` 時關閉縮放動畫。

## 7. 文字動畫

| 行為 | Class／機制 | 注意 |
|------|-------------|------|
| **區塊進場** | `.pe55-anim-onview` + `.pe55-anim-in` | 與第 6 節圖組相同 IO 邏輯 |
| **主視覺文字階梯** | `.pe55-reveal` 包主視覺；子元素加 `.pe55-hero-line`，依序淡入上移 | 不與整區 `.pe55-anim-onview` 疊用於同一區，以免重複隱藏 |
| **段落／標題階梯** | 區塊內包 `.pe55-stagger`，腳本為每個子元素設 `--pe55-d` 延遲 | 僅在該區塊已 `.pe55-anim-in` 後播放 |
| **無 JavaScript** | 未加 `pe55-js-motion` 時不隱藏內容 | 與先前約定相同 |
| **減少動態** | `@media (prefers-reduced-motion: reduce)` 關閉動畫與圖片 hover 縮放 | 區塊 `transition` 一併關閉 |

**原則**：動畫僅點綴；更複雜動效可改 **GIF／短影片**（權衡檔案大小與後台上限）。

---

## 8. 與 HTML 的對應方式

- 設計 token 集中在 **`#pe55-cms-safe-root`**，勿在官網主題容器上掛 class。
- 新增區塊時：優先使用本檔之 **class 名稱與色票變數**，必要新樣式請寫入**同一** `<style>` 底部並註解分段。
- 西文裝飾句：`<p class="pe55-latin" lang="es">Perú</p>` 這類寫法。
